## Power Network System (PNS) Control
We consider a PNS composed of several power generation area coupled through tie-lines. Tie lines allow for power exchange between area. In this way, if the load profile of an area grows too much with respect to the power generated in that area neighbouring areas can contribute to satisfy the request.
The objective is to balance and control thos power exchange to satisfy energy request.

🔗 Repository:
https://github.com/LoreB43/Networked-Control
